# drimian-web

Especificacion completa para el sitio web de Drimian (drimian.com).

## Estructura

| Archivo | Que contiene |
|---------|-------------|
| `WEB DRIMIAN - Documento Maestro.md` | Especificacion completa: arquitectura, copy, diseno, implementacion |
| `WEB DRIMIAN - Diagnostico Interactivo.md` | Las 10 preguntas del diagnostico + scoring + narrativas (Capa 1: mecanismos) |
| `WEB DRIMIAN - Mecanica Diagnostico Capa 2.md` | Matriz mecanismo-indicador + algoritmo + radar + narrativas por perfil (Capa 2: prosperidad) |
| `WEB DRIMIAN - Benchmarking Consultoras.md` | Analisis de Cognizant, McKinsey, Bain, BCG, Accenture + recomendaciones |

## Concepto

La web NO es un folleto. Es una experiencia interactiva donde el dueno de empresa:

1. Responde 10 preguntas sobre como hace las cosas (mecanismos que producen prosperidad)
2. Ve como eso impacta en 7 indicadores de prosperidad (radar visual)
3. Descubre las 2-3 palancas de mayor impacto para su empresa
4. Decide contactar por WhatsApp

## Stack sugerido

- HTML/CSS/JS puro o Next.js
- Chart.js para el radar de 7 indicadores
- Tally (gratis) para MVP del assessment, o ScoreApp/Interact para version profesional
- Vercel o Netlify para hosting
- Peso total de pagina < 1MB
